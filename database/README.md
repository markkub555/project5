# Database

ไฟล์ในโฟลเดอร์นี้ใช้สำหรับเริ่มต้นฐานข้อมูลของโปรเจกต์

- `init/01-schema.sql` สร้างตารางหลักที่ระบบใช้ทันทีเมื่อรัน Docker บนฐานข้อมูลใหม่

หมายเหตุ:

- ไฟล์ใน `database/init/` จะถูกรันอัตโนมัติโดย MySQL container เฉพาะตอน volume ยังว่าง
- ถ้าต้องการรัน schema ใหม่อีกรอบ ให้ลบ volume เดิมก่อนแล้วค่อย `docker compose up -d --build`
